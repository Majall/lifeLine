LIFELINE
Harmony in Red
------------------------------------------------------------------------------------------

Welcome to Lifeline. A website merely created with the intention of attracting more people towards donating blood. This website created with HTML, CSS , JavaScript and PHP only. Website has a database and can create  update delete retrieve data using SQL. Logo was designed using illustrator and Images are found through the Internet. When a person who is interested in donating blood enters the web application, they can find guide, on how blood donating process happens and what are the requirements. If a Guest wants to make an appointment. they could simply make it. but to view the history of the appointments they should be registered into the system. Then the appointments are managed by the relevant hospitals. The validity of a Hospital is managed by the System Admin.
